HTML

HyperText Markup Language

Es el lenguaje de marcado utilizado para la creación y estructuración de contenido en páginas web. Define la jerarquía y organización de elementos como encabezados, párrafos, enlaces, imágenes y otros, utilizando etiquetas.

CSS

Cascading Style Sheets

Es un lenguaje de estilo utilizado para describir la presentación y el formato de un documento HTML. CSS permite controlar aspectos como colores, fuentes, márgenes, alineaciones y diseño, separando así el contenido estructural del diseño visual.

JS

Client-Side Scripting Language

Es un lenguaje de programación que se utiliza para agregar interactividad y dinamismo a las páginas web. Permite manipular el contenido de la página, interactuar con el usuario, realizar peticiones al servidor y actualizar la interfaz de usuario en tiempo real.
